Habitican Randomizer

This tool will let you randomize stuff in your Habitica account.

Questions, suggestions, feature requests, bug reports, can be done in the GitHub Repository or in the Habitica Guild.

To use this tool you need to provide your UUID and API key, which you can find at Settings->API on the website, or at Settings->API on the app. This tool will not memorize your API, but you can use your browser's password managment tool to not have to write it each time.
